# LexAI - AI-Powered Legal Document Reviewer

LexAI is an AI-driven web application that helps users analyze legal documents intelligently. It allows users to upload `.txt` or `.pdf` files, and then performs tasks like summarizes the content in the file uploaded, clause detection, and risk evaluation â€” all powered by AI. Finally, it generates a downloadable PDF report with all the key insights.

---

## Features

- Upload and read `.txt` and `.pdf` legal documents.
- Gives a short summary of the content.
- Automatically detect key legal clauses.
- View a risk score based on the document.
- Download a report as a PDF.
